Phasmophobia Helper

Phasmophobia is a game in which you hunt ghosts with a group of friends. It is unique due to how which ghost will spawn is chosen. Each ghost has a unique combination of 3 pieces of evidence which identifies it. Figuring out which evidence is present on the current map determins which ghost you think it is.

In order to help us note down which evidence we currently have, we created Phasmophobia Helper. You are able to click on the evidences you have, and it will tell you which ghosts it could be given that combination. It also handles any mutual exclusions with evidences in the game. For example, having "Ghost Orbs" will automatically disable "Ghost Writing" as only one of them can be present.

For the design of the website, we went for Neumorphism, which is where the buttons extrude from the page - rather than just plonking them ontop the background. Also, there is a toggle for light and dark mode, aswell as a "simple" mode which removes the text from the buttons. This gives the user full customizability of the website.
